
# Pareidolia Mobile

>A React Native application used to take videos for dataset building to be sent to the [Pareidolia Desktop](https://github.com/Strawberry64/Pareidolia-Desktop) app.

---

## Overview

Pareidolia Mobile is a multi-platform mobile application that is used for building datasets for user personal dataset building for image classification model training, building, and fine-tuning.

## Features
- Send images and video to Pareidolia Desktop
- Collect video data for model training
- Cross-platform support (iOS, Android)

## Installation

1. **Clone the repository**
       ```sh
       git clone https://github.com/Strawberry64/Pareidolia-Mobile
       ```
2. **Navigate to the project directory**
       ```sh
       cd pareidolia-mobile
       ```
3. **Install dependencies**
       ```sh
       npm install
       ```
4. **Prebuild**
       ```sh
       npx expo prebuild
       ```
5. **Run on Development Build**
       ```sh
       npx expo run:ios
       OR
       npx expo run:android
       ```

## Usage

1. Start the mobile app
2. Create and set your target dataset profile
3. Record desired videos of objects for particular dataset
4. Connect to Pareidolia Desktop application
5. Organize and upload videos to desktop application
6. Load model profiles for desired inferencing
